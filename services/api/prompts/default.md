Preservation Rules

Furniture and Object Handling
Keep all existing large furniture in place; maintain exact position, scale, and color.

Straighten existing frames, lampshades, mirrors, and decor in-place only.

Declutter

Remove personal identifiers: family photos, mail, certificates, names, addresses, badges.

Mirrors and Reflections

Windows and Exterior Views

If the exterior view has recoverable detail, gently recover it (up to 1-2 stops) while preserving what is truly visible.

Maintain exterior exposure 0.5-1 stop darker than interior when detail exists; otherwise allow natural highlight roll-off.

Photographic Realism
Maintain real photographic texture: natural grain, material imperfections, fabric wear, surface variation.
Preserve true depth of field and natural shadow roll-off.
Light must read as originating from existing windows and practical lights already present.
